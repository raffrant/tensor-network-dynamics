# Project Notes

## Main themes

### Measurement-based concentration
The Python code indicates a workflow where controlled-phase resource states are transformed by local measurements and corrective rotations, then optimized against a target state through fidelity-based loss functions.

### Time-crystal dynamics
The driven Ising notebook uses disorder, near-\(\pi\) spin rotations, and repeated gate application while tracking an alternating local magnetization signal, making it a natural starting point for discrete time-crystal studies.

### Heisenberg-chain evolution
The Heisenberg notebook uses two-site gates and long-time evolution of an MPS, then records middle-site magnetization as a simple local observable.
